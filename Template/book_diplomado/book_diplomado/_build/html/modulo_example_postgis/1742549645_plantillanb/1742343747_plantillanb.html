Caso de estudio: Administraci√≥n de datos para generaci√≥n de modelo clasificador de Bosque / No Bosque por medio de PostGIS y Python#

El caso de estudio ‚ÄúBosque - No Bosque‚Äù consiste en la clasificaci√≥n de √°reas geogr√°ficas en dos categor√≠as: regiones con cobertura boscosa y regiones sin cobertura boscosa. Este an√°lisis es fundamental para la gesti√≥n ambiental, la planificaci√≥n territorial y el monitoreo del cambio clim√°tico. Para lograr esta clasificaci√≥n, se integran datos r√°ster y vectoriales almacenados en una base de datos PostgreSQL, combinados con t√©cnicas de Machine Learning en Python.

El proceso comienza con la extracci√≥n de datos r√°ster almacenados en PostGIS, los cuales contienen informaci√≥n RGB de una imagen satelital. Luego, se combinan con muestras de entrenamiento vectoriales que indican la cobertura real de la tierra, permitiendo entrenar un modelo de clasificaci√≥n basado en Random Forest. Este modelo aprende a diferenciar √°reas boscosas y no boscosas a partir de las firmas espectrales de los p√≠xeles.

La integraci√≥n de PostGIS y Python es clave para procesar datos espaciales de manera eficiente. PostGIS permite realizar consultas geoespaciales avanzadas y gestionar datos r√°ster y vectoriales en una base de datos optimizada, mientras que Python proporciona herramientas avanzadas para el an√°lisis de datos y el entrenamiento de modelos de aprendizaje autom√°tico. La combinaci√≥n de ambas tecnolog√≠as permite automatizar la extracci√≥n de datos, la clasificaci√≥n y la visualizaci√≥n de los resultados en mapas interpretables.

Este enfoque facilita la toma de decisiones basadas en datos espaciales, permitiendo la identificaci√≥n de patrones de deforestaci√≥n, el monitoreo de la salud de los bosques y la generaci√≥n de pol√≠ticas ambientales m√°s efectivas.

Extraer los valores de p√≠xel de cada banda del r√°ster para cada muestra

# Listas para almacenar los valores de los p√≠xeles extra√≠dos y sus respectivas etiquetas
valores_pixeles = []
etiquetas = []
# Iteramos sobre cada fila del DataFrame que contiene las muestras de entrenamiento
for _, row in df.iterrows():
    lon, lat, label = row['lon'], row['lat'], row['class']                              # Extraemos las coordenadas y la clase de la muestra    
    # Convertimos coordenadas geogr√°ficas (lon, lat) a √≠ndices de p√≠xel en la imagen r√°ster
    row_idx, col_idx = ~transform * (lon, lat)                                          # Aplicamos la transformaci√≥n inversa
    row_idx, col_idx = int(row_idx), int(col_idx)                                       # Convertimos a enteros para obtener la posici√≥n en la matriz r√°ster
    # Verificamos que las coordenadas convertidas est√©n dentro de los l√≠mites del r√°ster
    if 0 <= row_idx < raster_array.shape[1] and 0 <= col_idx < raster_array.shape[2]:
        # Extraemos los valores de las bandas del p√≠xel correspondiente y los almacenamos en la lista
        valores_pixeles.append([
            raster_array[0, row_idx, col_idx],  # Banda 1 (Ej. Rojo)
            raster_array[1, row_idx, col_idx],  # Banda 2 (Ej. Verde)
            raster_array[2, row_idx, col_idx]   # Banda 3 (Ej. Azul)
        ])
        etiquetas.append(label)                                                         # Almacenamos la etiqueta de la muestra en la lista
# Convertimos las listas a arreglos NumPy para su uso en el modelo de Machine Learning
X, y = np.array(valores_pixeles), np.array(etiquetas)

Entrenamiento del modelo de clasificaci√≥n

# Inicializaci√≥n del modelo de clasificaci√≥n Random Forest
modelo = RandomForestClassifier(
    n_estimators=1000,      # N√∫mero de √°rboles en el bosque
    max_depth=10,           # Profundidad m√°xima de cada √°rbol para evitar sobreajuste
    max_features="sqrt",    # N√∫mero m√°ximo de caracter√≠sticas consideradas en cada divisi√≥n (ra√≠z cuadrada del total)
    random_state=42,        # Fijamos una semilla para asegurar reproducibilidad de resultados

# Realizaci√≥n de predicciones sobre el conjunto de prueba
